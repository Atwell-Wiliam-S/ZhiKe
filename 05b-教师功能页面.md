## 4.2.4 知识图谱管理页 (Teacher)

![知识管理页参考图](images/knowledge-manage.jpg)

### 布局结构
顶部筛选栏 + 左右分栏（左 40% 视频预览，右 60% 片段列表）。

| 区域 | 桌面端 (≥1024px) | 平板 (768–1023px) | 移动端 (<768px) |
| --- | --- | --- | --- |
| 筛选栏 | 全宽 | 全宽 | 全宽，筛选器竖排 |
| 视频预览 | 左 40%，Sticky | 左 35%，Sticky | 全宽，非 Sticky |
| 片段列表 | 右 60% | 右 65% | 全宽 |

### 顶部筛选栏
- **面包屑**：课程管理 > {课程名} > 知识片段，`--color-text-tertiary`，`<nav aria-label="面包屑">`
- **筛选器**：课程下拉 + 视频下拉 + 搜索框，`--color-bg-elevated` (#eeefe9) 背景，`--radius-sm`，gap 12px

### 左栏：视频预览
- **视频播放器**：16:9 比例，带缩略图，`--radius-md`
- **时间轴叠加**：视频下方显示片段分段色条 (高度 8px，`--radius-full`)
  - 绿色 (#008b00)：已确认
  - 琥珀色 (#F7A501)：待审核
  - 珊瑚色 (#fb565b)：需修改
- **Sticky**：`position: sticky; top: 80px`

### 右栏：片段卡片列表
- **卡片**：`--radius-sm` 圆角，`--color-border` 边框，padding 16px，margin-bottom 12px
- **内容**：
  - 标题：IBM Plex Sans 700, 21.4px，`--color-text-primary`
  - 时间范围：Source Code Pro 14px, `--color-text-secondary`（如 02:30 - 05:15）
  - 内容预览：3行截断，`--color-text-secondary`
  - 关键词：Pill 徽章，`--color-accent-subtle` 背景 + `--color-accent` 文字，gap 6px
  - 操作按钮：编辑 (Pencil)、预览 (Eye)、删除 (Trash2)，Dark Primary (#1e1f23) 样式
- **悬停**：`--color-bg-hover` 背景 + 文字闪现橙色 (#F54E00)

### 底部操作栏
- "添加片段" 按钮（Dark Primary 样式，Plus 图标）
- 批量操作（全选 Checkbox、批量删除、批量确认）

### 无障碍备注
- 面包屑使用 `<ol>` 有序列表 + `aria-label="面包屑"`
- 时间轴色条使用 `aria-label` 描述各颜色含义
- 片段卡片操作按钮 `aria-label="编辑片段"` / `删除片段`
- 批量操作使用 `aria-label="全选所有片段"`

## 4.2.5 学情分析大屏 (Teacher/Admin)

![学情分析大屏参考图](images/page-analytics-dashboard.jpg)

### 布局结构
顶部操作栏 + 2×2 Grid 图表区，padding 24px。

| 区域 | 桌面端 (≥1280px) | 桌面端 (1024–1279px) | 平板 (<1024px) |
| --- | --- | --- | --- |
| 图表网格 | 2×2 Grid | 2×2 Grid | 1×1 单栏堆叠 |
| 热力图 | 跨 2 列 | 跨 2 列 | 全宽 |
| KPI 卡片 | 1/3 宽 | 1/3 宽 | 全宽，2×2 Grid |

### 顶部操作栏
- **左侧**：课程选择下拉 (Select) + 日期范围选择器 (DatePicker)
- **右侧**：导出按钮（Dark Primary 样式，Download 图标）
- **背景**：`--color-bg-card`，`--radius-sm`，padding 16px，margin-bottom 20px

### 图表网格

#### 左上 (2/3)：难点热力图
- **类型**：ECharts 水平热力图 (`heatmap`)
- **容器**：`--color-bg-card` 背景，`--radius-sm`，padding 20px
- **标题**："视频难点热力图"，IBM Plex Sans 600, 16px
- **X 轴**：视频时间轴（00:00 → 10:00），格式 mm:ss
- **Y 轴**：学生密度（0% → 100%）
- **配色**：蓝色 (#3b82f6, 低) → 黄色 (#eab308, 中) → 红色 (#ef4444, 高) 渐变
- **标记**：难点峰值处红色脉冲圆点 (CSS animation, 2s)
- **交互**：悬停 Tooltip 显示 "时间: 03:25, 难度分数: 0.87, 困惑学生: 68%"

#### 右上 (1/3)：班级概览 KPI
- **容器**：`--color-bg-card` 背景，`--radius-sm`，padding 20px
- **4 个指标卡片**（2×2 Grid，gap 12px）：
  - 完成率：ECharts 环形进度图 (`gauge`) + 百分比数字
  - 活跃学生：数字 + 环比趋势箭头 (TrendingUp/TrendingDown)
  - 平均观看时长：数字 + 时分秒格式 (如 1h 23m)
  - 总提问数：数字 + 环比趋势箭头
- **样式**：IBM Plex Sans 700 大数字 (28px)，`--color-text-primary`，`--color-border` 卡片边框

#### 左下 (1/2)：高频困惑词云
- **容器**：`--color-bg-card` 背景，`--radius-sm`，padding 20px
- **类型**：ECharts 词云图 (`wordCloud`)
- **标题**："高频困惑词云"，IBM Plex Sans 600, 16px
- **配色**：高频词 `--color-text-primary` (#4d4f46) 大字 (48-64px)，低频词 `--color-text-tertiary` (#9ea096) 小字 (14-20px)
- **交互**：悬停 Tooltip 显示 "词频: 23 次"

#### 右下 (1/2)：学生排行榜
- **容器**：`--color-bg-card` 背景，`--radius-sm`，padding 20px
- **类型**：ECharts 水平柱状图 (`bar`)
- **标题**："学习时长 Top 10"，IBM Plex Sans 600, 16px
- **内容**：Top 10 学生，头像缩略图 (24px 圆形) + 姓名 + 学习时长
- **配色**：橄榄墨 (#4d4f46) 柱条，`--radius-sm` 圆角
- **交互**：悬停 Tooltip 显示 "张三: 12h 35m, 完成率 89%"

### 无障碍备注
- 图表容器使用 `role="img"` + `aria-label` 提供文字摘要
- 热力图：`aria-label="难点热力图：共检测到 5 个难点区域，最高难度在 03:25"`
- 词云：`aria-label="高频困惑词：导数(23次), 极限(18次), 积分(15次)..."`
- 排行榜：下方提供 `<table>` 格式的可访问数据表
- 导出按钮 `aria-label="导出分析报告"`

## 4.2.7 视频上传页 (Teacher)

### 布局结构
单栏居中，最大宽度 800px，padding 32px 24px。

| 区域 | 桌面端 (≥768px) | 移动端 (<768px) |
| --- | --- | --- |
| 上传区域 | 400px 高，虚线边框 | 200px 高 |
| 表单 | 双列表单布局 | 单列堆叠 |

### 上传区域
- **拖拽区**：虚线 `--color-border` 边框 (2px dashed)，`--radius-lg` 圆角，padding 48px
- **内容**：居中
  - Upload 图标 (48px, `--color-text-tertiary`)
  - "拖拽视频文件到此处" (`--color-text-primary`, 16px 600)
  - "或点击选择文件" (`--color-accent`, 链接样式)
  - "支持 MP4/WebM/MOV, 最大 500MB" (`--color-text-tertiary`, 12px)
- **拖拽悬停**：`--color-accent-subtle` 背景，`--color-accent` 实线边框 (2px solid)，scale(1.01)

### 上传进度
- **进度条**：`--color-accent` 填充，`--radius-full`，高度 8px
- **状态文字**：上传中... 68% / 上传完成 ✓ / 上传失败 ✗
- **失败重试**：显示 "重新上传" 链接

### 视频信息表单
- **字段**：
  - 课程选择（Select 下拉，必填）
  - 视频标题（Input，必填，max 100 字符）
  - 视频描述（Textarea，可选，max 500 字符，字数计数器）
  - 排序序号（InputNumber，可选）
- **预览**：上传后显示视频缩略图预览 (16:9, `--radius-md`)
- **验证**：实时验证，必填字段标红星

### 操作按钮
- **布局**：右对齐，gap 12px
- "取消" 按钮（Ghost 样式，`--color-text-secondary` 边框）
- "保存并解析" 按钮（Dark Primary #1e1f23 样式，Sparkles 图标 + 文字）

### 无障碍备注
- 拖拽区使用 `<input type="file" accept="video/mp4,video/webm,video/quicktime">` 隐藏，点击触发
- 拖拽区 `role="button"` + `aria-label="点击或拖拽上传视频文件"`
- 进度条 `role="progressbar"` + `aria-valuenow` + `aria-valuemin/max`
- 表单验证错误 `aria-describedby` 关联到字段

## 4.2.8 用户管理页 (Admin)

![用户管理页参考图](images/page-user-manage.jpg)

### 布局结构
顶部操作栏 + 筛选栏 + 数据表格 + 分页。

| 区域 | 桌面端 (≥1024px) | 平板 (768–1023px) | 移动端 (<768px) |
| --- | --- | --- | --- |
| 表格 | 全宽，6 列 | 全宽，水平滚动 | 卡片列表模式 |
| 筛选栏 | 一行横排 | 一行横排 | 竖排堆叠 |

### 顶部操作栏
- **左侧**："用户管理" 标题 (system-ui 24px 700)
- **右侧**："添加用户" 按钮（Dark Primary 样式，UserPlus 图标）

### 筛选栏
- **角色筛选**：Pill 徽章切换（全部 / 学生 / 教师 / 管理员），`--radius-full`
- **搜索**：输入框 + Search 图标前缀，placeholder "搜索用户名或邮箱..."
- **状态筛选**：启用 / 禁用 Toggle 开关

### 数据表格
- **列**：用户名、邮箱、角色（Pill 徽章）、状态（Toggle 开关）、注册时间、操作
- **角色徽章颜色**：
  - 学生：`--color-info-bg` 背景 + `--color-info` 文字
  - 教师：`--color-success-bg` 背景 + `--color-success` 文字
  - 管理员：`--color-warning-bg` 背景 + `--color-warning` 文字
- **行**：padding 14px 16px，`--color-border` 底部分割线
- **行悬停**：`--color-bg-hover` 背景 + 文字闪现橙色 (#F54E00)
- **操作**：编辑 (Pencil)、删除 (Trash2, `--color-danger` 悬停，二次确认弹窗)

### 移动端卡片模式
- 每个用户一张卡片，包含：头像 + 用户名 + 角色 Pill + 状态 + 操作按钮
- 筛选栏竖排堆叠

### 分页
- **位置**：底部居中
- **样式**：页码按钮 + "每页 10 条" 下拉 + "共 156 条" 文字
- **当前页**：`--color-accent` 背景 + 白色文字

### 无障碍备注
- 表格语义化 `<table>/<caption>`
- 角色筛选使用 `role="tablist"` + `aria-selected`
- Toggle 开关 `role="switch"` + `aria-checked`
- 删除确认弹窗 `role="alertdialog"` + `aria-describedby`
- 分页 `<nav aria-label="用户列表分页">`
