# 4.1 设计系统规格 (DESIGN.md)

> 来源：智能教学闭环系统重构设计文档 v1.2
> 
> 本文档定义系统的完整设计系统，参考 Google Stitch DESIGN.md 规范 + awesome-design-md 设计令牌体系。核心理念：通过结构化文档定义 UI 规范，使 AI Agent 和开发者能像读取 API 文档一样理解设计约束，实现像素级一致的界面输出。

---

## 4.1.1 设计理念与原则

| 原则 | 说明 |
| --- | --- |
| **结构化可机器读取** | 所有设计令牌通过 CSS 变量定义，AI Agent 可直接解析 |
| **共享基底 + 角色主题** | 布局/间距/动效共享，颜色/字体/阴影按角色切换 |
| **暗色优先护眼** | 学生端默认暗色，长时间学习不疲劳 |
| **数据密度适配** | 学生端留白呼吸感，教师端信息密集专业感 |
| **无障碍合规** | WCAG 2.1 AA 级对比度，键盘可导航，语义化 HTML |

## 4.1.2 多角色设计风格策略

系统服务三种角色，采用"共享基底 + 角色主题"架构。

| 角色 | 设计风格参考 | 主题模式 | data-theme 值 | 适用场景 |
| --- | --- | --- | --- | --- |
| 学生 (STUDENT) | Linear + Notion | 暗色/亮色可选 | `student-dark` / `student-light` | 视频学习、AI助教、笔记 |
| 教师 (TEACHER) | PostHog | 浅色固定 | `teacher-light` | 知识片段管理、学情查看 |
| 管理员 (ADMIN) | PostHog | 浅色固定 | `teacher-light` | 用户管理、系统配置、日志 |

**主题切换逻辑：**登录后从 JWT `roleCode` 自动路由。学生端可通过顶部按钮手动切换暗色/亮色，偏好存入 `localStorage('zhike-theme')`。教师/管理端固定浅色，不可切换。

## 4.1.7 动效规范 (Motion)

### 过渡时长

| 变量 | 值 | 适用 |
| --- | --- | --- |
| `--duration-instant` | 0ms | 无动画（尊重 prefers-reduced-motion） |
| `--duration-fast` | 100ms | 按钮按下、开关切换 |
| `--duration-normal` | 200ms | 颜色变化、展开/收起 |
| `--duration-slow` | 300ms | 模态框出现、页面切换 |
| `--duration-slower` | 500ms | 复杂动画、骨架屏渐变 |

### 缓动函数

| 变量 | 值 | 适用 |
| --- | --- | --- |
| `--ease-default` | `cubic-bezier(0.4, 0, 0.2, 1)` | 标准过渡 |
| `--ease-in` | `cubic-bezier(0.4, 0, 1, 1)` | 元素进入 |
| `--ease-out` | `cubic-bezier(0, 0, 0.2, 1)` | 元素离开 |
| `--ease-in-out` | `cubic-bezier(0.4, 0, 0.2, 1)` | 双向动画 |

### 关键动画

| 动画 | 定义 | 适用 |
| --- | --- | --- |
| Fade In | `opacity 0→1, 200ms ease-out` | 页面加载、Toast |
| Slide Up | `translateY(8px)→0 + opacity 0→1, 200ms ease-out` | 下拉菜单 |
| Scale In | `scale(0.95)→1 + opacity 0→1, 200ms ease-out` | 模态框 |
| Pulse Glow | `box-shadow 0 0 0 0 var(--color-accent-subtle) → 0 0 0 8px transparent, 2s infinite` | 知识点标记 |
| Skeleton Shimmer | `background-position: -200% 0 → 200% 0, 1.5s infinite linear` | 骨架屏加载 |
| Spinner | `rotate(0→360deg), 1s linear infinite` | 按钮加载 |

### 无障碍动效

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 4.1.12 设计红线与禁忌 (Do's and Don'ts)

| 规则 | 说明 |
| --- | --- |
| ✓ 学生端暗色使用 `#1a1a2e` 主背景 | 深灰蓝对比度适中，护眼 |
| ✓ 学生端使用 `#6366f1` 静藤蓝作为暗色强调 | 柔和且专业 |
| ✓ 教师端使用 `#fdfdf8` 暖纸背景 | 不用纯白作为页面背景 |
| ✓ 教师端悬停时闪现橙色文字 | PostHog 风格核心交互 |
| ✓ 所有颜色通过 `var(--color-xxx)` 引用 | 禁止硬编码色值 |
| ✓ 文字对比度 ≥ 4.5:1 (AA) | 保护视力 |
| ✓ 动画尊重 `prefers-reduced-motion` | 无障碍合规 |
| ✗ 学生端暗色不要使用纯黑背景 | `#050507` 对比度过高 |
| ✗ 教师端不要使用重阴影 | 仅边框和表面色差创造层级 |
| ✗ 不要跨主题混用色彩令片 | 严格通过 CSS 变量切换 |
| ✗ 不要在教师端使用大圆角 (>8px) | 保持 4-6px 紧凑风格 |
| ✗ 不要使用纯色块作为唯一信息传达方式 | 必须配合文字/图标 |
| ✗ 不要在暗色主题上使用低对比度灰色文字 | 三级文字仅用于大字号 |

### useTheme Composable 完整实现

```typescript
// src/composables/useTheme.ts
import { ref, watchEffect } from 'vue'

type Theme = 'student-dark' | 'student-light' | 'teacher-light'

const STORAGE_KEY = 'zhike-theme'

const currentTheme = ref<Theme>('student-dark')

export function useTheme() {
  /** 初始化主题 */
  function initTheme(roleCode: string) {
    if (roleCode === 'STUDENT') {
      const saved = localStorage.getItem(STORAGE_KEY) as Theme | null
      if (saved && (saved === 'student-dark' || saved === 'student-light')) {
        currentTheme.value = saved
      } else {
        // 跟随系统偏好
        const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches
        currentTheme.value = prefersDark ? 'student-dark' : 'student-light'
      }
    } else {
      // 教师/管理员固定浅色
      currentTheme.value = 'teacher-light'
    }
    applyTheme()
  }

  /** 应用主题到 DOM */
  function applyTheme() {
    document.documentElement.setAttribute('data-theme', currentTheme.value)
  }

  /** 切换学生端暗色/亮色 */
  function toggleTheme() {
    if (currentTheme.value === 'student-dark') {
      currentTheme.value = 'student-light'
    } else if (currentTheme.value === 'student-light') {
      currentTheme.value = 'student-dark'
    } else {
      return // 教师端不可切换
    }
    localStorage.setItem(STORAGE_KEY, currentTheme.value)
    applyTheme()
  }

  /** 是否为暗色 */
  const isDark = ref(true)

  watchEffect(() => {
    isDark.value = currentTheme.value === 'student-dark'
  })

  // 监听系统主题变化（仅学生端未手动设置时）
  if (typeof window !== 'undefined') {
    window.matchMedia('(prefers-color-scheme: dark)')
      .addEventListener('change', (e) => {
        if (!localStorage.getItem(STORAGE_KEY)) {
          currentTheme.value = e.matches ? 'student-dark' : 'student-light'
          applyTheme()
        }
      })
  }

  return { currentTheme, isDark, initTheme, toggleTheme }
}
```
