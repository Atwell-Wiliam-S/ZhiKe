## 4.2.6 课程详情页 (Student)

![课程详情页参考图](images/page-course-detail.jpg)

### 布局结构
顶部面包屑 + Hero 区 + Tab 内容区 + 右侧边栏。

| 区域 | 桌面端 (≥1024px) | 平板 (768–1023px) | 移动端 (<768px) |
| --- | --- | --- | --- |
| Hero | 左图右文，2 列 | 左图右文，图 40% | 全宽堆叠，图在上 |
| Tab 内容 | 左 70% | 左 65% | 全宽 |
| 右侧边栏 | 右 30%，Sticky | 右 35%，Sticky | 隐藏，移至底部 |

### Hero 区域
- **左侧**：课程封面图，16:9 比例，`--radius-lg` 圆角，`--shadow-md` 阴影，`object-fit: cover`
- **右侧**：padding-left 32px
  - 课程标题：system-ui 28px 700，`--color-text-primary`
  - 教师信息：头像 (40px 圆形) + 姓名 + 职称，`--color-text-secondary`
  - 评分：⭐ 4.8 (156 评价)，`--color-warning` 星星
  - 统计：24 个视频 · 12 小时 · 156 名学生，`--color-text-secondary`，SeparatorDot 分隔
  - 课程描述：3行截断 + "展开全部" 链接 (`--color-accent`)
  - CTA 按钮："开始学习"（`--color-accent` 背景，`--color-accent-text` 文字，全宽，44px 高）

### Tab 导航
- **标签**：课程概览 | 课程目录 | 学员评价
- **样式**：下划线指示器 (2px `--color-accent`)，`--color-text-secondary` 默认态，`--color-text-primary` 激活态
- **动画**：指示器滑动 transition 200ms ease-out

### 课程目录 Tab
- **手风琴列表**：`<details>/<summary>` 原生折叠
  - 第一章：微积分基础（3 个视频，2 小时 15 分）
    - 1.1 函数与极限 (45:00) ✅ 已完成 (`--color-success`)
    - 1.2 导数与微分 (50:00) ▶ 正在学习 (`--color-accent` 脉冲)
    - 1.3 积分入门 (40:00) 🔒 未解锁 (`--color-text-tertiary`)
  - 第二章：线性代数...
- **当前学习项**：`--color-accent` 左边框 (3px) + 加粗 + `--color-accent-subtle` 行背景

### 右侧边栏 (Sticky)
- **课程统计卡片**：完成进度环形图 + 已学/总课时
- **教师简介卡片**：头像 + 姓名 + 职称 + 课程数
- **相关课程推荐**：3 个课程卡片缩略图

### 无障碍备注
- 面包屑 `<nav aria-label="面包屑"><ol>`
- Tab 使用 `role="tablist"` + `role="tab"` + `role="tabpanel"` + `aria-selected`
- 手风琴使用原生 `<details>/<summary>` 语义
- 视频状态图标使用 `aria-label`："已完成" / "正在学习" / "未解锁"
- CTA 按钮 `aria-label="开始学习：{课程名}"`

## 4.2.9 问答历史页 (Student)

![问答历史页参考图](images/page-qa-history.jpg)

### 布局结构
顶部筛选 + 问答卡片列表 + 分页。

| 区域 | 桌面端 (≥768px) | 移动端 (<768px) |
| --- | --- | --- |
| 筛选栏 | 一行横排 | 竖排堆叠 |
| 问答卡片 | 单列，max-width 800px | 单列全宽 |

### 筛选栏
- **搜索**：输入框 + Search 图标，placeholder "搜索问题内容..."
- **时间筛选**：Pill 徽章切换（今天 / 本周 / 本月 / 全部）
- **排序**：Select 下拉（最新优先 / 最早优先）

### 问答卡片列表
- **卡片**：`--color-bg-card` 背景，`--color-border` 边框，`--radius-md`，padding 20px，margin-bottom 12px
- **内容**：
  - 顶部：问题文本 (`--color-text-primary`, 600, 16px) + 时间戳 (`--color-text-tertiary`, 12px, 右对齐)
  - 中部：AI 回答预览（3行截断，`--color-text-secondary`, 14px）
  - 底部：flex 布局，gap 8px
    - 引用来源数量 Pill (`--color-accent-subtle` 背景 + `--color-accent` 文字，如 "3 个来源")
    - Token 消耗 (`--color-text-tertiary`, 如 "256 tokens")
    - 模型版本标签 (`--color-bg-elevated` 背景, 如 "GLM-4")
- **悬停**：边框变为 `--color-accent`，显示"查看详情" 链接按钮，transition 150ms
- **点击**：跳转到问答详情（或展开 Modal）

### 空状态
- 居中：MessageSquare 图标 (64px, `--color-text-tertiary`) + "暂无问答记录" + "去视频学习页提问" 按钮

### 无障碍备注
- 筛选 Pill `role="tablist"` + `aria-selected`
- 卡片使用 `<article>` 语义标签
- "查看详情" 链接 `aria-label="查看问答详情：{问题前20字}"`
- 空状态插图 `role="img" aria-label="暂无问答记录"`

### 亮色主题差异
- 背景 `#ffffff`，卡片 `#ffffff`，边框 `#e2e8f0`
- 强调色 `#0075de`，Pill 背景 `rgba(0,117,222,0.1)`

## 4.2.10 个人设置页 (All Roles)

![个人设置页参考图](images/page-settings.jpg)

### 布局结构
左侧导航 + 右侧设置表单，最大宽度 960px 居中。

| 区域 | 桌面端 (≥768px) | 移动端 (<768px) |
| --- | --- | --- |
| 设置导航 | 左侧 200px 固定 | 顶部水平 Tab |
| 设置表单 | 右侧自适应 | 全宽 |

### 左侧设置导航
- **样式**：`--color-bg-card` 背景，`--radius-md`，padding 8px
- **导航项**：
  - 个人信息 (User 图标)
  - 主题偏好 (Palette 图标，仅学生端显示)
  - 安全设置 (Shield 图标)
  - 通知设置 (Bell 图标)
- **激活态**：`--color-accent` 左边框 (3px) + `--color-accent-subtle` 背景 + `--color-text-primary` 文字
- **默认态**：`--color-text-secondary` 文字，悬停 `--color-bg-hover`

### 个人信息表单
- **头像区域**：
  - 圆形头像 (80px, `--radius-full`, `object-fit: cover`)
  - "更换头像" 按钮 (Ghost 样式，Camera 图标，在头像右下角叠加)
- **表单字段**：
  - 用户名 (Input, `readonly`, `--color-bg-elevated` 背景)
  - 邮箱 (Input, type="email")
  - 手机号 (Input, type="tel")
  - 个人简介 (Textarea, max 200 字符，字数计数器)
- **保存按钮**：`--color-accent` 背景，`--color-accent-text` 文字，右对齐

### 主题偏好（学生端）
- **选项**：Radio Group — 暗色模式 (Moon 图标) / 亮色模式 (Sun 图标) / 跟随系统 (Monitor 图标)
- **预览**：小型实时预览卡片 (200×120px)，展示选中主题下的卡片样式
- **切换**：选择后立即生效，`useTheme` composable 响应

### 安全设置
- **修改密码**：
  - 旧密码 (Input, type="password", Eye/EyeOff 切换)
  - 新密码 (Input, type="password")
  - 确认新密码 (Input, type="password")
- **密码强度指示器**：
  - 弱：红色进度条 (25%), "弱"
  - 中：黄色进度条 (50%), "中"
  - 强：绿色进度条 (100%), "强"
  - 规则：≥8 字符 + 大写 + 小写 + 数字 + 特殊字符

### 通知设置
- **选项**：Toggle 开关列表
  - 课程更新通知
  - 作业提醒
  - 系统公告
  - 邮件通知汇总

### 无障碍备注
- 设置导航使用 `<nav aria-label="设置导航">`
- 表单字段 `<label>` 关联 `for` 属性
- 密码强度 `role="progressbar"` + `aria-valuenow` + `aria-label="密码强度：中"`
- Toggle `role="switch"` + `aria-checked`
- 主题选项 `role="radiogroup"` + `aria-checked`

## 4.2.11 笔记列表页 (Student)

![笔记列表页参考图](images/page-notes-list.jpg)

### 布局结构
顶部筛选 + 笔记卡片瀑布流。

| 区域 | 桌面端 (≥1024px) | 平板 (768–1023px) | 移动端 (<768px) |
| --- | --- | --- | --- |
| 筛选栏 | 一行横排 | 一行横排 | 竖排堆叠 |
| 笔记卡片 | 2 列 Grid, gap 16px | 2 列 Grid, gap 12px | 1 列全宽 |

### 筛选栏
- **课程筛选**：Select 下拉，placeholder "选择课程"
- **视频筛选**：Select 下拉（联动课程，课程变更时重置），placeholder "选择视频"
- **搜索**：输入框 + Search 图标，placeholder "搜索笔记内容..."

### 笔记卡片
- **布局**：CSS Grid `grid-template-columns: repeat(auto-fill, minmax(320px, 1fr))`，gap 16px
- **卡片**：`--color-bg-card` 背景，`--color-border` 边框，`--radius-md`，padding 20px
- **内容**：
  - 顶部：视频标题 (`--color-text-primary`, 600, 14px) + 时间戳 Pill (indigo 背景, 白色文字, 如 [02:30])
  - 中部：笔记内容 (`--color-text-primary`, 14px, line-height 1.6)，完整显示
  - 底部：flex 布局，justify-between
    - 左：创建时间 (`--color-text-tertiary`, 12px)
    - 右：公开/私有 Toggle + 编辑 (Pencil) / 删除 (Trash2) 图标按钮
- **悬停**：`--shadow-md` 阴影 + translateY(-1px)，transition 150ms

### 空状态
- 居中：StickyNote 图标 (64px, `--color-text-tertiary`) + "暂无笔记" + "去视频学习页创建笔记" 按钮

### 无障碍备注
- 笔记卡片使用 `<article>` 语义
- 时间戳 Pill `aria-label="视频时间点 2分30秒"`
- 公开/私有 Toggle `role="switch"` + `aria-checked` + `aria-label="公开笔记"`
- 删除按钮二次确认 `role="alertdialog"`

### 亮色主题差异
- 背景 `#ffffff`，卡片 `#ffffff`，边框 `#e2e8f0`
- 时间戳 Pill `#0075de` 背景
- 悬停阴影 `rgba(0,0,0,0.08)`

## 4.2.12 通知中心 (All Roles)

### 布局结构
右侧滑出 Drawer 面板，宽度 400px（桌面端）/ 全宽（移动端）。

| 区域 | 桌面端 (≥768px) | 移动端 (<768px) |
| --- | --- | --- |
| 面板 | 右侧 Drawer, 400px | 全屏 Drawer |
| 触发 | 顶部导航铃铛图标 | 同左 |

### 面板头部
- **标题**："通知" (system-ui 18px 700)
- **操作**："全部已读" 按钮 (Ghost 样式) + 关闭按钮 (X 图标)

### 通知分类 Tab
- **标签**：全部 | 系统 | 课程 | 互动
- **样式**：Pill 徽章切换，`--radius-full`
- **未读计数**：Tab 右上角红色 Badge (如 "3")

### 通知列表
- **列表项**：padding 14px 16px，`--color-border` 底部分割线
- **未读样式**：左侧 3px `--color-accent` 边框 + `--color-accent-subtle` 行背景
- **已读样式**：默认背景
- **内容**：
  - 图标 (32px 圆形背景)：系统 (Info) / 课程 (BookOpen) / 互动 (MessageCircle)
  - 标题 (`--color-text-primary`, 14px 600) + 时间 (`--color-text-tertiary`, 12px)
  - 描述 (`--color-text-secondary`, 13px, 2行截断)
- **悬停**：`--color-bg-hover` 背景
- **点击**：跳转到对应页面并标记已读

### 空状态
- 居中：BellOff 图标 (48px) + "暂无通知"

### 无障碍备注
- Drawer `role="dialog"` + `aria-modal="true"` + `aria-label="通知中心"`
- 打开时 Trap Focus，Escape 关闭
- 通知列表 `role="list"` + `aria-label="{分类}通知列表"`
- 未读项 `aria-unread="true"`
- Tab 切换 `role="tablist"` + `aria-selected`

## 4.2.13 404 错误页 (Public)

### 布局结构
全屏居中，最小高度 100vh。

### 内容
- **背景**：`--color-bg-canvas`
- **插图**：大号 404 数字 (system-ui 120px 800, `--color-text-tertiary`, opacity 0.3) 或自定义插图
- **标题**："页面未找到" (system-ui 24px 700, `--color-text-primary`)
- **描述**："抱歉，您访问的页面不存在或已被移除。" (`--color-text-secondary`, 16px)
- **操作按钮**：
  - "返回首页" 按钮 (`--color-accent` 背景，`--color-accent-text` 文字，Home 图标)
  - "返回上页" 链接 (Ghost 样式，ArrowLeft 图标)
- **布局**：flex 居中，gap 16px

### 交互
- 按钮 hover：`--color-accent-hover` 背景
- 自动 10 秒后倒计时跳转首页（底部小字提示 "10s 后自动返回首页"）

### 无障碍备注
- `role="alert"` 标记错误状态
- 主内容区域 `role="main"`
- 按钮 `aria-label="返回首页"`
- 倒计时使用 `aria-live="polite"` 播报
