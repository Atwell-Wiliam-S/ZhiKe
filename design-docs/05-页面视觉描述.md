# 4.2 核心页面视觉描述 (Agent Prompt Guide)

> 来源：智能教学闭环系统重构设计文档 v1.2
>
> 本文档为 AI Agent 和前端开发者提供每个页面的精确视觉描述，可直接作为 Prompt 输入给 AI 生成 UI 代码，也可作为前端开发的像素级参考。所有页面遵循 `04-设计系统规格.md` 中定义的设计令牌。

## 通用规格

### 响应式断点

| 断点名 | 宽度范围 | 典型设备 | 布局策略 |
| --- | --- | --- | --- |
| `xs` | 0–639px | 手机竖屏 | 单栏堆叠，侧边栏收起为 Drawer |
| `sm` | 640–767px | 手机横屏 | 单栏堆叠，卡片 1 列 |
| `md` | 768–1023px | 平板 | 双栏适配，侧边栏可折叠 |
| `lg` | 1024–1279px | 小笔记本 | 标准双栏/三栏 |
| `xl` | 1280–1535px | 桌面 | `--container-2xl` (1280px) 居中 |
| `2xl` | 1536px+ | 大屏 | `--container-3xl` (1440px) 居中 |

### 页面转场动画

| 转场类型 | 动画 | 时长 | 缓动 | 适用场景 |
| --- | --- | --- | --- | --- |
| 页面切换 | `fade + slide-right` | 200ms | `ease-out` | 路由切换 |
| 模态弹窗 | `fade + scale(0.95→1)` | 150ms | `cubic-bezier(0.16,1,0.3,1)` | Dialog/Modal |
| Drawer 侧栏 | `translate-x(-100%→0)` | 250ms | `cubic-bezier(0.16,1,0.3,1)` | 移动端侧边栏 |
| Toast 通知 | `slide-down + fade` | 200ms | `ease-out` | 全局提示 |
| 卡片悬停 | `translateY(-2px) + shadow` | 150ms | `ease-out` | 功能卡片 |
| 列表项进入 | `fade + slide-up(8px)` | 200ms | `ease-out` (stagger 30ms) | 列表渲染 |

> **无障碍**：所有动画均尊重 `prefers-reduced-motion: reduce`，此时降级为 `opacity` 瞬切（0ms）。

### 通用无障碍规范

| 规范 | 实现要求 |
| --- | --- |
| **键盘导航** | 所有可交互元素支持 Tab 聚焦，Enter/Space 激活，Escape 关闭弹窗 |
| **焦点指示器** | `outline: 2px solid var(--color-accent); outline-offset: 2px` |
| **ARIA 标签** | 图标按钮需 `aria-label`，导航需 `aria-current="page"`，进度需 `aria-valuenow` |
| **语义化 HTML** | 导航用 `<nav>`，主内容用 `<main>`，侧边栏用 `<aside>`，表格用 `<table>` |
| **颜色对比度** | 文本与背景对比度 ≥ 4.5:1 (AA)，大文本 ≥ 3:1 |
| **屏幕阅读器** | 图表提供 `aria-label` 文字摘要，图标用 `role="img" + aria-label` |
| **跳过导航** | 页面顶部隐藏 "跳到主内容" 链接，Tab 聚焦时显示 |

## 4.2.0 登录/注册页 (Public)

![登录页参考图](images/page-login.jpg)

### 布局结构
左右分栏布局（桌面端），移动端堆叠。

| 区域 | 桌面端 (≥1024px) | 平板 (768–1023px) | 移动端 (<768px) |
| --- | --- | --- | --- |
| 左侧品牌区 | 50% 宽度，全高 | 40% 宽度 | 隐藏或顶部小 banner (120px) |
| 右侧表单区 | 50% 宽度，居中 | 60% 宽度 | 100% 宽度 |

### 左侧品牌区
- **背景**：深灰紫蓝 (#1a1a2e) → 静藤蓝 (#6366f1) 对角渐变 (135deg)
- **装饰**：浮动的抽象几何图形（圆、三角、线条），低透明度 (0.1-0.2)，CSS `@keyframes float` 缓动动画
- **内容**：居中偏上
  - Logo "ZhiKe" + 标语 "AI 驱动的智能教学闭环系统"
  - 3 个特性图标 + 简短描述（竖排）：
    - 🎓 多模态视频解析
    - 🤖 AI 智能问答助手
    - 📊 学情热力图分析

### 右侧表单区
- **背景**：`var(--color-bg-canvas)`（跟随系统主题）
- **内容**：垂直居中，最大宽度 400px，padding 32px
  - Logo 小图标（仅移动端显示）
  - 标题："欢迎回来" (登录) / "创建账号" (注册)，system-ui 24px 700
  - 副标题："登录以继续学习" / "开始你的学习之旅"，`--color-text-secondary`
  - 表单字段：
    - 用户名输入框（带 User 图标前缀）
    - 密码输入框（带 Lock 图标前缀 + 显示/隐藏切换 Eye/EyeOff 图标）
    - [注册页额外] 角色选择（学生/教师 Radio Group）
    - [注册页额外] 邮箱输入框
  - 记住我 Checkbox + 忘记密码链接（同一行，两端对齐）
  - 登录按钮：`--color-accent` 背景，`--color-accent-text` 文字，全宽，40px 高，`--radius-md`
  - 分割线："或" 文字 + 两侧横线
  - 第三方登录（预留，暂不实现）
  - 底部链接："没有账号？立即注册" / "已有账号？立即登录"

### 交互状态
| 状态 | 视觉反馈 |
| --- | --- |
| 输入框聚焦 | `--color-border-focus` 边框 + `--color-accent-subtle` 光环 (box-shadow) |
| 登录按钮悬停 | `--color-accent-hover` 背景，translateY(-1px) |
| 登录按钮按下 | scale(0.98)，`--color-accent` 加深 |
| 登录失败 | 输入框 `--color-danger` 边框 + 下方红色错误提示文字 |
| 登录中 | 按钮 Spinner + "登录中..." 文字，`pointer-events: none` |
| 表单验证 | 实时验证，输入框失焦时触发 |
| Enter 键 | 提交表单 |

### 无障碍备注
- 表单使用 `<form>` 标签，登录按钮 `type="submit"`
- 密码显示/隐藏按钮 `aria-label="显示密码" / "隐藏密码"`
- 错误提示关联 `aria-describedby` 到输入框
- 角色选择 Radio Group 用 `role="radiogroup"` + `aria-labelledby`

### 亮色主题差异
- 右侧背景 `#ffffff`，输入框背景 `#f8fafc`
- 强调色 `#0075de`，聚焦光环 `rgba(0,117,222,0.15)`
- 左侧品牌区渐变保持不变

## 4.2.1 学生首页 (Student Homepage)

![学生首页参考图](images/student-homepage.jpg)

### 布局结构
单栏滚动布局，最大宽度 `--container-2xl` (1280px) 居中，padding 0 24px。

| 区域 | 桌面端 (≥1024px) | 平板 (768–1023px) | 移动端 (<768px) |
| --- | --- | --- | --- |
| Top Nav | 64px 固定顶部 | 56px 固定顶部 | 56px 固定顶部 |
| Hero | 全宽，padding 48px 0 32px | padding 32px 0 24px | padding 24px 0 16px |
| 功能卡片 | 2×3 Grid，gap 24px | 2×3 Grid，gap 16px | 1×2 Grid，gap 12px |
| 最近活动 | 全宽列表 | 全宽列表 | 全宽列表 |

### Top Navigation
- **高度**：64px，`--color-bg-card` 背景，底部 `--color-border` 边框
- **左侧**：Logo "ZhiKe" + 品牌色圆点 (8px)
- **右侧**：主题切换按钮 (Sun/Moon 图标) + 通知铃铛 (Badge 红点) + 角色徽章 Pill + 用户头像 Dropdown

### Hero 区域
- **间距**：顶部 48px，底部 32px
- **问候语**："Hi, {username}"，system-ui 36px 400，`--color-text-primary`
- **副标题**："今天想学点什么？"，Inter 18px，`--color-text-secondary`

### 功能卡片网格
- **布局**：2×3 CSS Grid，gap 24px
- **卡片样式**：`--color-bg-card` 背景，`--color-border` 边框，`--radius-lg` 圆角，padding 24px
- **悬停**：边框变为 `--color-accent`，`--shadow-glow` 光晕，translateY(-2px)，transition 150ms ease-out
- **卡片内容**：
  1. 📚 我的课程 — Lucide `BookOpen` 图标(indigo) + 标题 + 描述 + 右箭头 `ChevronRight`
  2. ▶️ 继续学习 — Lucide `PlayCircle` 图标(indigo) + 标题 + 上次学习进度条 (thin, `--color-accent`)
  3. 🤖 AI 助教 — Lucide `Bot` 图标(indigo) + 标题 + "有问题随时问我"
  4. 📝 我的笔记 — Lucide `StickyNote` 图标(indigo) + 标题 + 笔记数量 Pill
  5. 📊 学习报告 — Lucide `BarChart3` 图标(indigo) + 标题 + 本周学习时长
  6. ⚙️ 设置 — Lucide `Settings` 图标(indigo) + 标题 + "主题、个人信息"

### 最近活动
- **标题**："最近活动"，system-ui 20px 600，margin-bottom 16px
- **列表**：时间线样式，每条包含：Lucide 活动图标 + 描述 + 相对时间 (`--color-text-tertiary`)
- **空状态**：居中插图 (Lucide `Inbox` 大图标, 64px, `--color-text-tertiary`) + "开始你的第一次学习吧" 文字

### 无障碍备注
- 功能卡片使用 `<ul role="list">` + `<li>` 语义列表
- 每张卡片为 `<a>` 或 `<button>`，包含 `aria-label` 描述完整功能
- 进度条使用 `role="progressbar"` + `aria-valuenow` + `aria-valuemin/max`
- 通知铃铛 Badge 使用 `aria-label="3 条未读通知"`

### 亮色主题差异
- 背景 `#ffffff`，卡片 `#ffffff`，边框 `#e2e8f0`
- 强调色 `#0075de`，光晕 `rgba(0,117,222,0.2)`
- Hero 问候语 `#0f172a`，副标题 `#64748b`

## 4.2.2 沉浸式视频学习页 (Student Video Learning)

![视频学习页参考图](images/video-learning.jpg)

### 布局结构
两栏布局，左侧 70% + 右侧 30%，全宽背景（无最大容器限制）。

| 区域 | 桌面端 (≥1024px) | 平板 (768–1023px) | 移动端 (<768px) |
| --- | --- | --- | --- |
| 整体布局 | 左 70% + 右 30% | 左 65% + 右 35% | 单栏堆叠，右栏折叠到底部 Tab |
| 视频播放器 | 16:9 比例 | 16:9 比例 | 16:9 比例，全宽 |
| AI 问答面板 | 左栏底部，max-h 400px | 左栏底部，max-h 300px | 全宽，max-h 50vh |
| 知识大纲 | 右栏 Sticky | 右栏 Sticky | 底部手风琴 Tab |

### 左栏 (70%)

#### 视频播放器
- **容器**：`--color-bg-card` 背景，`--radius-lg` 圆角，16:9 比例，`--shadow-md`
- **播放器**：原生 HTML5 Video + 自定义控件
- **控件栏**：底部半透明渐变叠加层 (rgba(0,0,0,0.6))，播放/暂停、进度条、音量、全屏按钮
- **快捷键**：Space 暂停/播放，←→ 快退/快进 5s，↑↓ 音量，F 全屏

#### 知识点时间轴
- **位置**：视频播放器正下方，margin-top 12px
- **样式**：水平时间轴，总长度 = 视频时长，高度 32px
- **标记**：indigo (#6366f1) 圆点 (6px) 标记知识点位置，悬停显示知识点摘要 Tooltip
- **当前进度**：`--color-accent` 实线 (3px) + 圆形拖拽手柄 (12px)
- **脉冲动画**：当前播放位置 ±30s 内的标记有 `--shadow-glow` 脉冲效果 (2s 循环)

#### AI 问答面板
- **容器**：`--color-bg-card` 背景，`--radius-lg` 圆角，padding 20px
- **标题栏**："AI 助教" + 在线状态指示灯(绿色 8px 圆点) + 清空对话按钮 (Trash2 图标)
- **消息列表**：可滚动区域，max-height 400px，gap 12px
  - 用户消息：右对齐，`--color-accent-subtle` 背景，`--radius-md` 圆角，max-width 80%
  - AI 消息：左对齐，`--color-bg-elevated` 背景，左侧 2px `--color-accent` 边框，max-width 85%
  - 引用标签：`--color-accent` 背景 + 白色文字的 Pill (8px 12px)，可点击跳转到视频时间点
  - 流式输出：闪烁光标 `▊` 动画 (1s blink)
- **输入栏**：底部固定，gap 8px
  - 多行 `<textarea>`（自动增高，min 40px，max 120px / 4 行）
  - 发送按钮：`--color-accent` 圆形按钮 (36px)，发送中显示 Spinner
  - Enter 发送，Shift+Enter 换行

### 右栏 (30%)

#### 知识大纲
- **标题**："知识大纲" + ChevronDown 折叠图标
- **列表**：手风琴样式 (`<details>/<summary>`)
  - 章节 → 知识点（带时间戳 [02:30-05:15]，Source Code Pro 字体）
  - 当前播放项：`--color-accent` 左边框 (3px) + 脉冲圆点 + 加粗文字
  - 悬停：显示摘要 Tooltip
- **Sticky**：`position: sticky; top: 80px`

#### 我的笔记
- **标题**："我的笔记" + Plus 添加按钮
- **列表**：时间倒序，gap 8px
  - 每条笔记：时间戳 Pill (indigo) + 内容预览（2行截断，`text-overflow: ellipsis`）
  - 悬停：显示完整内容 Tooltip + 编辑/删除操作 (Pencil, Trash2 图标)

### 无障碍备注
- 视频播放器：`role="application"` 避免屏幕阅读器干扰，提供快捷键说明
- 时间轴标记：`role="slider"` + `aria-label="知识点: {title}"` + `aria-valuenow={seconds}`
- AI 消息列表：`role="log"` + `aria-live="polite"` 实时播报新消息
- 引用标签：`aria-label="跳转到 {time} - {title}"`
- 输入框：`aria-label="向 AI 助教提问"`

### 亮色主题差异
- 全宽背景 `#ffffff`，卡片 `#ffffff`，边框 `#e2e8f0`
- 引用标签 `#0075de` 背景
- 时间轴标记 `#0075de`
- AI 消息左边框 `#0075de`
