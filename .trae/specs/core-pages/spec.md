# 核心页面开发 - 产品需求文档

## 概述
- **Summary**：基于05a-通用规格与核心页面.md文档，实现登录/注册页面、学生首页、视频学习页面和教师首页的前端开发。
- **Purpose**：构建智能教学闭环系统的核心用户界面，提供良好的用户体验和无障碍支持。
- **Target Users**：学生、教师和管理员用户。

## 目标
- 实现响应式登录/注册页面，支持表单验证和主题切换
- 实现学生首页，包括功能卡片网格和最近活动时间线
- 实现沉浸式视频学习页面，包括视频播放器、知识点时间轴和AI问答面板
- 实现教师首页，包括KPI卡片和课程管理表格
- 确保所有页面符合无障碍规范
- 实现页面转场动画和交互效果

## 非目标（范围外）
- 实现教师功能页面（如知识图谱管理、学情分析等）
- 实现学生通用页面（如课程详情、问答历史等）
- 实现后端API接口
- 实现真实的用户认证系统

## 背景与上下文
- 本项目基于Vue 3 Composition API和TypeScript开发
- 设计系统已在04-设计系统规格.md中定义
- 项目使用Vite作为构建工具，Pinia作为状态管理，Vue Router作为路由管理
- 参考design-md文件夹中的设计风格

## 功能需求
- **FR-1**：登录/注册页面 - 实现左右分栏布局，支持响应式设计，包含表单验证和错误处理
- **FR-2**：学生首页 - 实现顶部导航栏、Hero区域、功能卡片网格和最近活动时间线
- **FR-3**：视频学习页面 - 实现两栏布局，包含视频播放器、知识点时间轴和AI问答面板
- **FR-4**：教师首页 - 实现左侧侧边栏、KPI卡片和课程管理表格
- **FR-5**：响应式设计 - 支持不同屏幕尺寸的布局适配
- **FR-6**：主题切换 - 支持亮色和暗色主题
- **FR-7**：无障碍支持 - 实现键盘导航、屏幕阅读器支持等

## 非功能需求
- **NFR-1**：性能 - 页面加载时间小于2秒，交互响应时间小于100ms
- **NFR-2**：可维护性 - 代码结构清晰，模块化设计，遵循最佳实践
- **NFR-3**：可访问性 - 符合WCAG 2.1 AA级标准
- **NFR-4**：兼容性 - 支持主流浏览器（Chrome、Firefox、Safari、Edge）

## 约束
- **Technical**：Vue 3, TypeScript, Vite, Pinia, Vue Router
- **Business**：遵循设计系统规格，保持视觉一致性
- **Dependencies**：使用Lucide图标库，响应式设计框架

## 假设
- 前端项目基础结构已搭建完成
- 设计系统令牌已定义
- Mock数据已准备就绪

## 验收标准

### AC-1：登录/注册页面
- **Given**：用户访问登录/注册页面
- **When**：在桌面端查看
- **Then**：页面显示左右分栏布局，左侧为品牌区，右侧为表单区
- **Verification**：`human-judgment`

### AC-2：登录/注册页面响应式
- **Given**：用户访问登录/注册页面
- **When**：在移动端查看
- **Then**：页面显示单栏布局，左侧品牌区隐藏或变为顶部小banner
- **Verification**：`human-judgment`

### AC-3：学生首页功能卡片
- **Given**：学生用户登录系统
- **When**：访问学生首页
- **Then**：页面显示6个功能卡片，包含图标、标题和描述
- **Verification**：`human-judgment`

### AC-4：视频学习页面布局
- **Given**：学生用户访问视频学习页面
- **When**：在桌面端查看
- **Then**：页面显示两栏布局，左侧为视频播放器和AI问答面板，右侧为知识大纲和笔记
- **Verification**：`human-judgment`

### AC-5：教师首页KPI卡片
- **Given**：教师用户登录系统
- **When**：访问教师首页
- **Then**：页面显示4个KPI卡片，包含总学生数、活跃课程、平均完成率和今日提问
- **Verification**：`human-judgment`

### AC-6：无障碍支持
- **Given**：用户使用键盘导航
- **When**：访问任何页面
- **Then**：所有可交互元素支持Tab聚焦，Enter/Space激活，Escape关闭弹窗
- **Verification**：`programmatic`

### AC-7：主题切换
- **Given**：用户点击主题切换按钮
- **When**：在任何页面
- **Then**：页面主题在亮色和暗色之间切换
- **Verification**：`human-judgment`
